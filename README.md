# Arizona Nutrition Aid
## Nutritional Sciences & Wellness Foods Lab

### **Overview**
The **Arizona Nutrition Aid** is a Clinical Decision Support (CDS) tool developed for **The University of Arizona**. It utilizes the **Mission Loop** (Pattern / Rule / Solve) to automate clinical math and documentation for RDN candidates.

---

### **Clinical Authority (SSoT)**
* **Fiber Density:** 14g per 1000 kcal per the **U.S. Dietary Guidelines for Americans (DGA)**.
* **Ideal Body Weight:** Calculated via the **Hamwi Method**.

---

### **Integrated Tools**
* **Assessment Box:** IBW calculation (Metric/Imperial).
* **Nutrient Auditor:** Real-time fiber density auditing against DGA standards.
* **ADIME Box:** Automated PES statement generator with "Copy to Clipboard" functionality.

---

### **Clinical Disclaimer**
> **IMPORTANT:** This tool provides AI-enhanced clinical drafts. **Users are responsible for the final verification of all results.** All metrics must be audited against the primary SSoT (DGA/Hamwi) before use in a clinical setting.
